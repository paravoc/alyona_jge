📥 Торрент для новичков

🧲 Что это?

Торрент — это не файл, а способ обмена кусочками данных между людьми (P2P). Ты скачиваешь фильм не с одного сервера, а одновременно у сотни таких же, как ты.

📊 Почему «ест» трафик?

Пока ты скачиваешь — ты одновременно отдаёшь (раздаёшь) куски другим. Трафик летит в обе стороны. Провайдер видит нагрузку как загрузку + отдачу.

💡 Зачем придумали?

Чтобы разгрузить серверы. Чем больше людей качает — тем быстрее всем (в отличие от обычных серверов). Придумал Брэм Коэн в 2001 году, чтобы делиться файлами без перегрузок.

🌐 IP-адреса & подсети

🧩 IP устройства / сети

192.168.1.37 — это адрес твоего компа внутри сети. IP сети: 192.168.1.0 (адрес всей локальной сети).

IP-адрес уникальный номер устройства в сети (как индекс в доме) Маска подсети отделяет, какая часть относится к сети, какая — к устройству. Например 255.255.255.0 означает: первые 3 цифры — сеть, последняя — хост.

🔢 Что означают цифры?

IPv4 — 32 бита, записывается четырьмя десятичными числами (0–255). Маска в двоичном виде — это единицы (сеть) и нули (хост). /24 = 24 единицы, маска 255.255.255.0.

📟 Зачем IPv4 и IPv6?

IPv4 закончились (около 4.3 млрд адресов). IPv6 (например 2001:db8::1) — адресов хватит на каждую песчинку. Нужен для роста интернета, умных лампочек, холодильников и т.д.

🐍 Структуры Python

  • list = [1, 2, 3] — упорядоченный, можно менять, индекс 0..n.
  • set = {1, 2, 3} — неупорядоченный, только уникальные значения (быстрая проверка вхождения).
  • dict = {'key': 'value'} — пары ключ-значение, хеш-таблица.
  • defaultdict — из collections: словарь с значением по умолчанию (например список).
  • tuple = (1, 2, 3) — как список, но неизменяемый (быстрее, может быть ключом dict).
  • class — пользовательский тип: объединяет данные и методы. Чертеж для объектов.

💡 Пример: from collections import defaultdict
d = defaultdict(list) — автоматически создаёт пустой список для отсутствующего ключа.

🧩 Детали торрента

⚙️ .torrent файл содержит хеши (отпечатки) кусков. Клиент (qBittorrent) сверяет их. Раздача — это когда у тебя скачано 100% и ты отдаёшь другим. Рейтинг = отношение отданного/скачанного.

🔄 Магнет-ссылки — то же самое, без файла, просто хеш.

💬 маленький лайфхак

Если нужно быстро зашарить файл по сети — используй PowerToys + Locksmith (узнать, какая программа заняла файл) или Win+Shift+T чтобы выдернуть текст из фото.

🔬 Маска подсети — как это выглядит

IP: 192.168.1.145 маска: 255.255.255.0

В двоичном виде: 255 = 11111111. Значит первые 24 бита (192.168.1) — это сеть, последние 8 бит — устройство (145). Маска определяет границу.

Если маска 255.255.254.0 — это уже 23 бита, тогда в сети может быть 512 адресов (вместо 256). Цифры маски говорят, сколько бит «зарезервировано под сеть».

🗂️ Tuple против List

list: изменяемый (можно добавить, удалить). Занимает чуть больше памяти, медленнее.

tuple: неизменяемый (константа). Можно использовать как ключ в словаре. Быстрее работает.

set не хранит порядок, но проверка x in set — почти мгновенно (O(1)).

defaultdict очень удобен для подсчётов (частот) или группировки.
