📊 Задача 1. Кратные 5
📘 Теория: кратность 5 → число % 5 == 0. Используем list comprehension для фильтрации.
Условие: Найти максимальное число, кратное 5, также вычислить сумму, количество и среднее для всех кратных 5. Гарантируется хотя бы одно число кратное 5. 
a = []  # пустой список
n = int(input("Кол-во чисел: "))
for _ in range(n):
    x = int(input())
    if x % 5 == 0:
        a.append(x)

max_val = max(a)
summa = sum(a)
count = len(a)
average = summa / count

print(f"Максимум: {max_val} | Сумма: {summa} | Кол-во: {count} | Среднее: {average:.2f}")

🏆 Задача 3. 3 max и 2 min
📘 Теория: сортировка или nlargest/nsmallest. Список a содержит все числа.
Условие: Вывести 3 максимальных и 2 минимальных элемента последовательности (в порядке убывания/возрастания). Гарантируется минимум 5 чисел. 
a = []
n = int(input())
for _ in range(n):
    a.append(int(input()))

a_sorted = sorted(a)
two_min = a_sorted[:2]
three_max = a_sorted[-3:][::-1]  # в порядке убывания

print("3 максимальных:", three_max)
print("2 минимальных:", two_min)

🚗 Задача 4. Камера наблюдения
📘 Теория: среднее арифметическое, условие на количество автомобилей со скоростью ≤ 40.
Условие: Средняя скорость всех авто (с точностью 0.1) + вывести YES если не менее двух авто двигались со скоростью ≤ 40 км/ч, иначе NO. 
a = []
n = int(input())
for _ in range(n):
    a.append(int(input()))

avg = sum(a) / len(a)
count_slow = len([x for x in a if x <= 40])
verdict = "YES" if count_slow >= 2 else "NO"
print(f"{avg:.1f}\n{verdict}")

🔢 Задача 5. Семиричная система → окончание на '1'
📘 Теория: перевод в 7-ричную систему: остаток от деления на 7 даёт последнюю цифру. Условие: num % 7 == 1.
Условие: Найти сумму элементов последовательности, запись которых в 7-ричной системе счисления оканчивается на цифру 1. 
a = []
n = int(input())
for _ in range(n):
    val = int(input())
    if val % 7 == 1:   # последняя цифра в 7-ричной = 1
        a.append(val)

result_sum = sum(a)
print(result_sum)
