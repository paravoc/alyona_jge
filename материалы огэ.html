🧩 Задание 2: Формула включений-исключений (2 круга)

Запросы: пирожное | выпечка = 14200, пирожное = 9700, пирожное & выпечка = 5100. Найти: выпечка

🧮 |A ∪ B| = |A| + |B| – |A ∩ B|
14200 = 9700 + |выпечка| – 5100 ⇒ |выпечка| = 14200 – 9700 + 5100 = 9600
✔️ m(выпечка) = 9600 тыс.
📐 Формула для двух кругов всегда:
B = (A|B) - A + (A&B)
🔄 Задание 3: Три круга (полное пересечение)

Условие: Математика & Информатика = 330, Математика & Физика = 270, Математика & (Информатика | Физика) = 520.
Найти: Математика & Информатика & Физика

🔵 N4+N5 = 330
🟠 N5+N6 = 270
🟣 N4+N5+N6 = 520
Решаем: (330-N5) + N5 + (270-N5) = 520 → 600 - N5 = 520 → N5 = 80

🔢 Задание 5: Сумма цифр в восьмеричной записи (Python-метод)

Условие: Даны числа: 86₁₀, 99₁₀, 105₁₀. Найти число, сумма цифр которого в восьмеричной записи наименьшая.
В ответе записать эту сумму цифр.

📌 Ручной разбор:
86₁₀ → 8²=64, остаток 22, 8¹=8 → 2*8=16, остаток 6 → 86 = 1*64 + 2*8 + 6 = 126₈, сумма цифр = 1+2+6 = 9
99₁₀ → 64+35, 35=4*8+3 → 99 = 1*64 + 4*8 + 3 = 143₈, сумма=1+4+3=8
105₁₀ → 64+41, 41=5*8+1 → 105 = 1*64 + 5*8 + 1 = 151₈, сумма=1+5+1=7
✅ Минимальная сумма = 7 (у числа 105)

🐍 Python-инструмент (bin, oct, hex, int)

👉 Результат появится здесь
# Команды Python:
num = 105
oct_num = oct(num) # '0o151'
digits_sum = sum(int(d) for d in oct(num)[2:])
print(f"{num} в 8-ной = {oct_num}, сумма = {digits_sum}")

💡 Как решать: переводим число в восьмеричную систему (делением на 8), складываем цифры, сравниваем. Либо используем oct() в Python.

📐 Обобщение: Формулы и приёмы для ОГЭ

🔹 Для двух множеств

🔹 Для трёх множеств (общий случай)

|A ∪ B ∪ C| = |A|+|B|+|C| – |A∩B| – |A∩C| – |B∩C| + |A∩B∩C|
